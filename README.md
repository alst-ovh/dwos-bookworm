# dwos-bookworm

dwOS Bookworm Repository.

## Add Repository

```bash
curl -sSL https://alst-ovh.github.io/dwos-bookworm/setup_alst-ovh.sh | bash
```

## Install dwsetup

```bash
# install dwsetup
apt install dw-setup

# run dwsetup
dwsetup
```
